## Curso de React

OBS: ALGUNS CODIGOS FORAM MELHORADOS PARA CUSTOM DESIGN E adaptações

Criar projeto com:

```
npx create-react-app hello-world
```

Recomendado: Criar projeto com vite (https://vite.dev/):

```
npm create vite@latest
```

Outros comandos:

```
docker-compose up --build
```

```
npm run dev
```

```
npm run server
```

```
npm i tailwindcss @tailwindcss/vite framer-motion lucide-react
```
